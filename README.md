# 留学生ごみ分別サポート 最終展示版

## 実行方法
```cmd
npm install
npm run dev
```

## 主な改善点
- 100種類の不重複ごみデータ
- 分別検索と分別辞典を別機能化
- 日本語・中国語・英語UI切替
- お気に入り保存
- アンケート保存
- 統計画面
- SDGsプロジェクト説明
- Vite / React 対応

## 修正内容
- 收藏功能已修复：收藏时保存完整对象，避免数据更新后ID匹配失败。

## 收藏页UI修正
- 收藏页改成左侧列表 + 右侧详情。
- 增加查看搜索页按钮和移除收藏按钮。
- 避免点击收藏卡片直接误删。

## 最終修正
- 搜索结果变化时，右侧详情会自动同步为第一个搜索结果。
